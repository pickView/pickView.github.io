介绍

值类型：值类型是通过复制来传递的，这意味着它们在传递时会创建一个原始值的副本
引用类型：引用类型是通过引用传递的，这意味着多个变量可以引用同一个实例

Swift中Dictionary与NSDictionary异同

类型系统：

Dictionary 是 Swift 中的原生数据类型，属于值类型（value type）。它使用 Swift 的类型系统，并且在 Swift 代码中直接使用。
NSDictionary 是 Objective-C 中的类，属于引用类型（reference type）。它使用 Objective-C 的类型系统，在 Swift 代码中可以通过桥接使用（因为 Swift 可以与 Objective-C 进行交互）。
可选类型：

在 Swift 的 Dictionary 中，键和值都可以是可选类型。这意味着你可以通过键来查找值，但可能会得到 nil，表示没有对应的值。
在 Objective-C 的 NSDictionary 中，键和值可以是任意的 Objective-C 对象，包括 nil。
类型注解：

在 Swift 中，你需要明确指定字典的键和值的类型。例如：Dictionary<String, Int> 表示键为字符串，值为整数的字典。
在 Objective-C 中的 NSDictionary 中，键和值可以是任何 Objective-C 对象。
语法：

Swift 的 Dictionary 使用 [Key: Value] 的语法，如 [String: Int]。
Objective-C 的 NSDictionary 使用 @{key: value} 的语法，如 @{@”name”: @”John”}。
总之，虽然 Dictionary 和 NSDictionary 在某些方面类似，但它们是不同的类型，适用于不同的编程环境。在 Swift 代码中，推荐使用 Swift 的 Dictionary，因为它更符合 Swift 的类型系统和语法，并提供了更多的类型安全和功能。
