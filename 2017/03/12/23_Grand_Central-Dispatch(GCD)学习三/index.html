创建Dispatch Queue队列

系统提供的标准的Dispatch Queue

Serial Dispatch Queue(串行队列):Main Dispatch Queue

Concurrent Dispatch Queue(并行队列):Global Dispatch Queue

名称 Dispatch Queue 种类 优先级
Main Dispatch Queue Serial Dispatch Queue 主线程执行
Global Dispatch Queue(High Priority) Concurrent Dispatch Queue 高
Global Dispatch Queue(Low Priority) Concurrent Dispatch Queue 默认
Global Dispatch Queue(Background Priority) Concurrent Dispatch Queue 后台

多线程数据竞争问题

  • 多个线程更新相同资源导致数据竞争时使用Serial Dispatch Queue

在访问数据库和文件时，使用Serial Dispatch Queue可避免数据竞争的问题

读取处理追加到 Concurrent Dispatch Queue，写入处理在任一个读取处理没有执行的状态下，追加到 Serial Dispatch Queue 即可（在写入处理结束之前，读取处理不可执行）
