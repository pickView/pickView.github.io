GCD API

  • dispatch_set_taget_queue 指定线程的优先级
  • dispatch_barrier_async 屏障 之前的任务结束，才能执行以下任务
  • dispatch_after 定时器
  • dispatch Group 线程组
  • dispatch_apply 指定的次数将指定的block追加到指定的queue中
  • dispatch_suspend 挂起指定的queue
  • dispatch_resume 恢复指定的queue
  • dispatch_semaphone 持有计数的信号
  • dispatch_once 执行一次
