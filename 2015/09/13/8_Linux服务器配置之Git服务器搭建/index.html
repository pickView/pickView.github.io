2015-09-13 | Linux

一、配置环境

1、服务器：阿里云CentOS 7.2（64位） + Git （version 1.8.3.1）
2、客户端：Windows 10 Pro（64位） + Git（version 2.11.0.windows.1）
3、服务器连接软件：Xshell 5

  • 5、客户端创建 SSH 公钥和私钥

    此处我是管理了多个ssh_key，所以这边给公私钥起了个名字：id_rsa_git，生成后需要进行ssh-add操作，如何管理多个ssh_key可以参考：Git安装及SSH Key管理之Windows篇

    此时 C:\Users\用户名.ssh 下会多出两个文件 id_rsa 和 id_rsa.pub
    id_rsa_git 是私钥
    id_rsa_git.pub 是公钥
